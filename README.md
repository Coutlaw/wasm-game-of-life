### 🛠️ Build with `wasm-pack build`

```
wasm-pack build
```

### 🔬 Run with the JS submodule

```
cd www
git clone https://github.com/Coutlaw/wasm-life-site.git
mv ./wasm-life-site/* ./
npm install
npm run start
```
